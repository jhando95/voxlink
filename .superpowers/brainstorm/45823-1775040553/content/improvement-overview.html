Voxlink v0.9.3 — Polish & Refinement Batch

Curated improvements across visual polish, UX, and functional completeness

Visual Polish

1

Reply Quote Blocks

Replace inline reply text with visual quote blocks — left accent border, muted background, sender name + preview. Like Discord's reply rendering.

2

Animated Typing Indicator

Three bouncing dots animation when someone is typing, replacing static "User is typing..." text. Smooth 1.2s loop.

3

Status Dots on Members

Colored dot overlay on avatars — green (online), yellow (idle), red (DND), gray (offline/invisible). Consistent across member list, chat, and friend list.

4

Forwarded Message Styling

Small "↪ Forwarded from #channel-name" tag above forwarded messages in a muted accent color, visually distinct from replies.

UX Refinements

5

Collapsible Channel Categories

Click category headers to collapse/expand channel groups. Arrow indicator rotates on toggle. Collapse state persisted in config.

6

Disconnect Banner

Prominent banner at top when connection is lost — "Connection lost — Reconnecting..." with animated indicator. Replaces subtle status text.

7

Search Result Context

Show channel name badge on each search result. Bold the matching query text. Show timestamp + sender for context.

8

Keyboard Navigation

Arrow keys to navigate channel list. Enter to join. Escape to go back. Alt+Up/Down to switch channels quickly.

Functional Completeness

9

Thread View Panel

Side panel that opens when clicking a reply chain. Shows the parent message + all replies in a threaded view. Close with X or Escape.

10

Attachment Download Cards

File attachments render as styled cards with filename, size, and download icon. Click to save. Distinct from regular message text.

11

Voice Quality Badges

Small badge on voice channels showing quality level (Low/Standard/High/Ultra). Color-coded: gray/blue/green/gold.

12

Unread Pulse Animation

Subtle pulse glow on space icons and channel badges when new unread messages arrive. Single pulse, not continuous — draws eye without annoying.
